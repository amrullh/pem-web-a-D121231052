🖥️ Web Hosting

Rumah Digital Tempat Website Anda Tinggal.

Alamat vs. Rumah Sebenarnya

Hosting (Web Hosting) adalah layanan yang menyediakan ruang penyimpanan (storage) dan sumber daya komputasi di server agar website Anda dapat diakses oleh siapa saja di internet, kapan saja.

Jika Nama Domain adalah Alamat rumah Anda (misal: `Jalan Mawar No. 10`), maka Hosting adalah Tanah dan Bangunan fisiknya. 🏠

Server hosting pada dasarnya adalah komputer super kuat yang selalu menyala (24/7) dan terhubung ke internet. Saat seseorang mengunjungi "alamat" (domain) Anda, server inilah yang "membukakan pintu" dan "menyajikan" file-file website Anda (HTML, CSS, gambar) ke browser pengunjung.

Ada berbagai jenis hosting, bayangkan seperti properti:

🏘️ Shared Hosting: Paling murah. Anda berbagi satu server besar dengan ratusan situs lain. Ibarat tinggal di kamar kos; berbagi dapur dan kamar mandi.

🏢 VPS (Virtual Private Server): Tingkat menengah. Anda masih berbagi server fisik, tapi Anda mendapatkan "unit apartemen" virtual Anda sendiri. Lebih aman dan terkontrol.

🏰 Dedicated Server: Paling mahal. Anda menyewa satu server fisik utuh. Ibarat memiliki seluruh gedung atau istana sendirian. Kontrol penuh!
