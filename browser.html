🌐 Web Browser

Jendela Ajaib Anda Menuju Dunia Web.

Penerjemah Kode Menjadi Visual

Browser (Web Browser) seperti Chrome, Firefox, atau Safari, adalah aplikasi perangkat lunak yang bertugas mengambil semua potongan (HTML, CSS, JS, gambar) dan merakitnya menjadi halaman web visual yang bisa Anda lihat dan gunakan.

Browser BUKAN internet. Ia adalah alat untuk "menerjemahkan" internet. Saat Anda mengunjungi sebuah situs, browser melakukan proses ajaib yang disebut Critical Rendering Path (Jalur Perenderan Kritis):

  1. Mengambil (Fetch): Browser meminta file `index.html` dari server (menggunakan HTTP dan IP yang didapat dari DNS).
  2. Mem-parsing HTML (DOM): Browser membaca HTML baris per baris untuk membangun struktur "pohon" yang disebut DOM (Document Object Model). Ini adalah cetak biru konten.
  3. Mem-parsing CSS (CSSOM): Saat browser menemukan link CSS, ia mengambil file itu dan membangun "pohon" lain yang disebut CSSOM (CSS Object Model). Ini adalah cetak biru *gaya* (warna, ukuran, posisi).
  4. Render Tree 🌳: Browser menggabungkan DOM dan CSSOM untuk membuat Render Tree. Pohon ini hanya berisi apa yang akan *benar-benar* ditampilkan di layar.
  5. Tata Letak (Layout): Browser menghitung dengan tepat di mana setiap elemen harus ditempatkan (koordinat X/Y) dan seberapa besar ukurannya (lebar/tinggi). Bayangkan seperti menyusun denah.
  6. Melukis (Paint) 🎨: Akhirnya! Browser mengambil denah tadi dan "melukis" piksel-piksel ke layar Anda.

Dan JavaScript? ⚡ Browser memiliki JavaScript Engine (seperti V8 di Chrome) untuk menjalankan kode JavaScript. Ini adalah bagian yang membuat website menjadi interaktif—mengklik tombol, memvalidasi formulim, memuat data baru tanpa me-refresh halaman (ini adalah inti dari React!).
